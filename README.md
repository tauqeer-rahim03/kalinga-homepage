## ğŸ“± Features

-   **Mobile-First Architecture:** Designed primarily for mobile experiences with touch-friendly sliders and optimized layouts, scaling up elegantly to desktop.
-   **Custom Grid Systems:** Uses CSS Grid and Flexbox to manage complex layouts (Bestsellers, Collections, Reviews).
-   **Native Scroll Snapping:** Horizontal scrolling sections on mobile (Reviews, Gifting, Promos) use CSS Scroll Snap for a native app-like feel without heavy JavaScript libraries.
-   **Semantic HTML5:** Clean, accessible markup structure.
-   **Interactive Elements:**
    -   Functional Category & Filter tabs.
    -   Search bar focus effects.
    -   Color swatch selection logic.
    -   Hover states for buttons and product cards.
-   **Responsive Components:**
    -   **Hero Section:** Full-screen split layout on desktop vs. text overlay on mobile.
    -   **Navigation:** Scrollable pill menu on mobile vs. standard links on desktop.
    -   **Conditional Rendering:** "Social UGC" section appears only on mobile; "Recently Viewed" layout adapts contextually.
